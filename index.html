A retro VB.NET console game to train users on handling order failures in your exchange app: missing mappings, GLNs, SupplierSpecific IDs, contacting customers, and navigating boarding sheets / SharePoint.

How to play

  1. Install the .NET SDK.
  2. Clone this repository:
git clone https://github.com/beedeetestacc-arch/order-failure-arcade.git
cd order-failure-arcade
dotnet run
